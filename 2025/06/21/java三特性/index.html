Java面向对象基础：继承、封装、多态

发表于 2025-06-21 分类于 技术 本文字数： 477 阅读时长 ≈ 2 分钟

Java 是一种面向对象的编程语言，其核心思想可以归结为三个关键词：继承（Inheritance）、封装（Encapsulation） 和 多态（Polymorphism）。掌握这三大特性，是理解 Java 编程范式的关键。

继承（Inheritance）

继承是 Java 支持代码复用的重要机制。子类可以继承父类的属性和方法，从而避免重复代码。使用关键字 extends 实现。

封装（Encapsulation）

封装指的是将类的属性私有化（private），通过 getter 和 setter 方法对外提供访问和修改的接口。它有助于控制访问权限，提高代码安全性和可维护性。

优点包括：

  • 限制对内部数据的非法访问；
  • 提高代码模块的独立性；
  • 方便调试和维护。

多态（Polymorphism）

多态分为编译时多态（方法重载）和运行时多态（方法重写）。在运行时多态中，父类引用指向子类对象时，可以根据实际对象调用对应的方法。

多态的优势：

  • 提高程序的可扩展性；
  • 降低耦合度；
  • 有利于面向接口编程。

总结

继承、封装、多态是 Java 面向对象思想的三大支柱，三者协同，使得 Java 程序具有良好的结构和可维护性。掌握这些基础特性，是深入学习框架和设计模式的前提。
